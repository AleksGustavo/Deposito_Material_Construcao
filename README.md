# Sistema de Gestão para Depósito de Material de Construção

## 📌 Descrição
Sistema web desenvolvido com Django e MongoDB para gestão de estoque, vendas e financeiro de múltiplas lojas.

## 🎯 Objetivo
Resolver problemas de controle manual, integrando estoque e vendas entre unidades.

## 👥 Usuários
- Dono da empresa
- Funcionários

## 🧱 Tecnologias
- Python
- Django
- MongoDB
- Docker
